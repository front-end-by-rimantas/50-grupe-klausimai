## Kaip galima stringa panaudoti objekto parametro iskvietimui:

```js
function test(a, b) {
    const d = a;
    const e = b;
    const f = { a: 'Jonas', b: 'Petras', c: 'Antanas' };
    const g = { a: 'Jonaitis', b: 'Petraitis', c: 'Antanaitis' };
    return f.d, g.e;
}
console.log(test('a', 'b'));
```
